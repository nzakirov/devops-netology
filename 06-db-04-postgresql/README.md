Управляющие команды для:

- вывода списка БД:
```
 \l[+]   [PATTERN]      list databases
```
- подключения к БД:
```
  \c[onnect] {[DBNAME|- USER|- HOST|- PORT|-] | conninfo}
                         connect to new database (currently "postgres")
```
- вывода списка таблиц:
```
\dt[S+] [PATTERN]      list tables
```
